# Sublime Text Configuration

This repo contains my favorite sublime text 3 configuration packages and settings.

Usage:

1. Install Sublime Text 3
2. Install Sublime Package Manager [https://packagecontrol.io/installation](https://packagecontrol.io/installation)
3. Install the following packages
      * AdavancedNewFile  
      * Alignment
      * Bracket Highlighter
      * Color Highlightet
      * CSS Format
      * DocBlockr
      * Git-StatusBar
      * Git
      * GitGutter
      * GitOpenChangedFiles
      * HighlightWords
      * HTMLBeautify
      * Javascript Beautify
      * JSON Reindent
      * KeyboardSpellCheck
      * LESS
      * Package Control
      * Pretty JSON
      * Side Bar Git
      * Sublime Bookmarks
      * Trailing Spaces
      * Trimmer
      * WordHighlight
      * AutoSpell
      * JSHint Gutter
4. Copy content inside User folder to ~/Library/Application Support/Sublime Text 3/Packages/User
